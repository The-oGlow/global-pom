## Installation

Install [maven](https://maven.apache.org/install.html)

## Usage

To use, set in your pom.xml:

```
    <parent>
        <groupId>com.glowa.maven.parent</groupId>
        <artifactId>global-pom</artifactId>
        <version>0.1.1-SNAPSHOT</version>
    </parent>
```
